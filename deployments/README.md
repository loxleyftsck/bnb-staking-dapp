# Deployment Artifacts Directory

This directory stores deployment information for different networks.

## Files

Each deployment creates a JSON file with network name and timestamp:

```
<network>-<timestamp>.json
```

**Example:**

```json
{
  "network": "bscTestnet",
  "chainId": 97,
  "deployer": "0x...",
  "timestamp": "2025-12-26T15:00:00.000Z",
  "contracts": {
    "MyToken": {
      "address": "0x...",
      "name": "HeraldToken",
      "symbol": "HLD"
    },
    "StakingPool": {
      "address": "0x...",
      "apr": 1000
    }
  }
}
```

## Networks

- `hardhat` - Local development
- `bscTestnet` - BSC Testnet (Chain ID: 97)
- `bsc` - BSC Mainnet (when ready)

**Note:** Files in this directory are auto-generated by deployment scripts.
